## 验收准备任务

### 任务 1: 环境部署验证

**目标**: 验证 Server + Frontend + Agent 可以正常部署和运行

**步骤**:

1. **Server 部署**

   ```bash
   cd /root/workspace/es-agents/server
   ./mvnw clean package -DskipTests
   java -jar target/quarkus-app/quarkus-run.jar
   ```

   - [ ] Server 启动成功
   - [ ] API 端点可访问 (http://localhost:8080)
   - [ ] 数据库连接正常

2. **Frontend 部署**

   ```bash
   cd /root/workspace/es-agents/frontend
   npm install
   npm run dev
   ```

   - [ ] Frontend 启动成功
   - [ ] 页面可访问 (http://localhost:3000)
   - [ ] 登录功能正常

3. **Agent 部署**
   ```bash
   cd /root/workspace/es-agents/agent
   go build -o esa-agent
   ./esa-agent --server-url http://localhost:8080
   ```

   - [ ] Agent 编译成功
   - [ ] Agent 注册到 Server
   - [ ] 心跳正常

**验收标准**:

- Server、Frontend、Agent 均可正常启动
- 三者之间通信正常
- 无严重错误日志

### 任务 2: 测试数据准备

**目标**: 准备演示所需的测试数据

**数据清单**:

   - 关联主机、版本、状态（在线/离线）

   - 自定义脚本

4. **部署目标** (至少 2 个)
   - 目标名称、描述、关联命令模板

5. **批量任务** (至少 1 个历史任务)
   - 展示批量执行历史记录

6. **系统事件日志** (至少 10 条)
   - 不同类型的事件（INFO、WARN、ERROR）

**数据准备方式**:

- 方式 A: 手动通过前端页面创建
